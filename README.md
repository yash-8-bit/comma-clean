## A Mini Csv File Cleaner Web App

This Web app project is used to clean the csv and tranform the value with basic functions.

**Tech Stack**
- FastApi
- Pandas
- Html,Css and JavaScript


**How To Use**

1. Clone the repo
```
git clone https://github.com/yash-8-bit/comma-clean
```
2. Enter in Directory
```
cd comma-clean
```
3. Download Libraries
```
uv sync 
```
4. Run the App
```
uv run fastapi dev
```
